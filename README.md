# PianoTrainer

Aplikacja webowa wspomagająca naukę gry na instrumencie klawiszowym z wykorzystaniem urządzeń MIDI.

## Technologie

- React
- Node.js
- Express
- PostgreSQL
- Web MIDI API
- VexFlow

## Uruchomienie

### Backend

```bash
cd backend
npm install
node src/server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Baza danych

Należy utworzyć bazę PostgreSQL o nazwie:

```sql
pianotrainer
```
